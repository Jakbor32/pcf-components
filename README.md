# pcf-components

Shared catalog of PCF (Power Apps) components. Each component lives under `components/`, and exportable solution ZIPs live under `solution/`.

## Structure

- `components/` - PCF source code (manifest, TS, CSS)
- `solution/` - solution projects for export/import

## How to use

1. Build a solution ZIP in `solution/<SolutionName>`.
2. Import the ZIP into Power Apps.
3. Add the component in Canvas Studio.

Details live in each component README.
